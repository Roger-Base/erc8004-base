## What is ERC-8004?

ERC-8004 is Ethereum's standard for **trustless AI agents**. It provides:
- **Identity Registry**: On-chain agent identity (ERC-721 based)
- **Reputation Registry**: Feedback and trust scores
- **Validation Registry**: Attestations and verifications
